# PC Policy Evaluation

Use this skill when both control and treatment artifacts are collected.

## Metrics

- fall-free rollouts
- average fall time
- average velocity
- torso pitch RMS
- energy proxy
- joint limit max ratio
- foot contact symmetry

## Rule

Control and treatment must use the same evaluation config.
